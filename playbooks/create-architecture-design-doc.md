# 操作手册：生成 Word 架构设计文档

## 适用场景

需要将项目架构、部署方式、验证结果、风险和下一步行动整理为正式 Word 文档。

## 操作步骤

1. 梳理读者、目标和详细程度。
2. 先生成 SVG/PPT 架构图。
3. 使用 `sips` 将 SVG 转为正确比例 PNG。
4. 将图嵌入 Word，竖页或横页按用户要求选择。
5. 文档中包含当前状态矩阵、风险表和下一步行动。
6. 打开 Word 并让用户确认优化点。

## 注意事项

- 避免使用 QuickLook 生成最终图片，可能导致比例异常。
- 文档中不要写入明文密钥。
- 保留 PPT/SVG/PNG 图源。
