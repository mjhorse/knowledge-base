# 本地知识库

这是一个本地优先的 Markdown Wiki，用于沉淀日常工作要点、项目阶段成果、概念知识、排障经验、决策记录和可复用操作手册。

## 设计原则

- Markdown 作为知识源格式，便于迁移、搜索和 AI 读取。
- 本地优先，后续可通过 Git、对象存储、向量库或云端 Wiki 扩展。
- 目录按知识用途拆分：时间线、项目、概念、决策、操作手册、排障、附件。
- 每阶段成果尽量结构化录入，保证长期可查、可复用。
