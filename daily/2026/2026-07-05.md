## 关键结论

- 目录结构按知识用途划分：daily、projects、concepts、decisions、playbooks、troubleshooting、assets、templates。
- 第一阶段先使用 Markdown 文件和 Claude Code 搜索，不急于引入向量库。
- 后续可增加 `knowledge-wiki-capture` 与 `knowledge-wiki-query` skills。
- Obsidian Git 的远程仓库需要在 Git 仓库层配置，插件本身只调用 Git 命令。
- `Auto commit interval` 是本地提交频率，`Auto push interval` 是推送到 GitHub 的频率。
- 日常建议使用 `Obsidian Git: Sync` 进行重要变更后的手动同步。
- Google OAuth Client 只代表应用身份；Gmail API 权限由授权请求中的 scopes 与用户 consent 决定，不在 OAuth Client 页面维护固定 API 权限清单。
- `gogcli` 长期保存 refresh token，access token 通常运行时临时换取；`tokeninfo?access_token=` 只能验证 access token，不能直接验证 refresh token。
- OpenClaw 通过飞书远程发 Gmail 的关键前提是：本机 OpenClaw 使用同一 macOS 用户、`gog auth list` 能看到授权账号、Google API 调用带 M78 代理、命令加 `--no-input` 避免交互等待。
- `gog auth keyring keychain` 可将 token 存储固定到 macOS Keychain，避免 `auto` 后端在后台触发 passphrase/file keyring 密码提示。
